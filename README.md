## Descripción

El proyecto consiste en el desarrollo de un Sistema de Gestión de Libros Electrónicos que permitirá administrar usuarios, libros, autores, categorías y préstamos virtuales de manera organizada y segura.

## Objetivo General

Desarrollar un Sistema de Gestión de Libros Electrónicos que permita administrar recursos bibliográficos digitales, usuarios y préstamos virtuales de manera eficiente, organizada y segura.
